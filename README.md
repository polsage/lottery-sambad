# Lottery Sambad

A modern lottery result website with an admin panel.

## Features

- Today's Result
- Previous Results
- Search Results
- Responsive Design
- Admin Dashboard
- Secure Login
- Add/Edit/Delete Results

## Tech Stack

Frontend
- React + Vite
- Tailwind CSS

Backend
- Node.js
- Express

Database
- PostgreSQL (Supabase)

Deployment
- Frontend: GitHub Pages
- Backend: Render
